# Curse of Strahd — DM Resource Expansion

This repository organizes immersive tools to run the **Curse of Strahd** campaign with expanded narration, NPC dossiers, encounters, atmosphere guides, and campaign trackers. All files are Markdown and sorted into thematic folders for quick table use.

## Structure Overview
- **01_Narration_And_Dialog/** Scene-by-scene prompts with pacing, body language, and branching guidance.
- **02_NPC_Dossiers/** Expanded cast profiles with social checks and voice notes.
- **03_Monsters/** Tactics, motivations, and terrain notes for early foes.
- **04_Encounters/** Modular encounters with outcomes and player agency paths.
- **05_Lookup_Tables/** Rumors and quick-insert details.
- **06_Campaign_Trackers/** Session log templates and pacing reminders.
- **07_Campaign_Notes/** Quickstart notes and cross-references.
- **08_Roleplaying_Guides/** Tone and voice aids for the DM.
- **09_Foreshadowing_Lore_Themes/** Recurring motifs to weave through play.
- **10_Weather_Atmosphere_Soundscapes/** Environmental and audio cues.
- **11_Items_And_Treasure/** Handouts and minor rewards.
- **12_Maps_Pins_And_Area_Files/** Map pin lists for scene placement.
- **13_Strahd_Encounters/** Optional and scheduled Strahd interactions.

Begin with `01_Narration_And_Dialog/00_Mists_To_Barovia.md` to guide the party’s first steps into Barovia.
